# AUDIT V4-13 — route-web-guards.js 분리

## 작업 목적

PWA V4-12 기준으로 `patches.js`에 남아 있던 비교적 독립적인 웹사이트 스와이프 및 경로 복원 보조 로직을 별도 파일로 분리했습니다.

이번 단계는 기능 변경이 아니라 구조 분리입니다.

## 새 파일

- `js/route-web-guards.js`

## 분리한 내용

- 웹사이트 목록 좌우 스와이프 탭 이동 보조
- 웹사이트 스와이프 시 시각 피드백 연결
- 길찾기/경로삭제 후 노란 도착지 마커 복귀 보조
- `resetRoute` 래핑 보조
- 경로 시트 닫힘 감지 후 인포카드/마커 복귀 보조
- `window.oaiResetRouteThenClose` 보조
