길찾기 자동차 경로가 지도에 표시되려면 카카오 REST API 키가 들어간 Cloudflare Worker가 반드시 정상 배포되어 있어야 합니다.

현재 앱 설정 URL:
https://kakao-rest-proxy.bong0219.workers.dev

필수 확인:
1. Cloudflare Worker가 실제로 배포되어 있어야 합니다.
2. Worker Secret 이름은 반드시 KAKAO_REST_API_KEY 여야 합니다.
3. 값은 Kakao Developers의 REST API 키입니다. JavaScript 키가 아닙니다.
4. Worker 코드는 kakao-rest-proxy-worker.js 파일 내용을 사용하세요.
5. 브라우저에서 아래 주소를 열었을 때 JSON routes가 나오면 정상입니다.

https://kakao-rest-proxy.bong0219.workers.dev?endpoint=directions&origin=128.591,35.871&destination=128.589,35.88&priority=RECOMMEND

이 Worker가 실패하면 앱 파일을 아무리 고쳐도 지도 안 자동차 경로는 표시되지 않습니다.
