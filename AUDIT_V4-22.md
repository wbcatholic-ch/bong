# AUDIT V4-22 — 웹사이트 CSS 분리 1단계

## 기준
- 기준 파일: PWA V4-21
- 작업 목표: 웹사이트(가톨릭 정보) 전용 CSS를 `style.css`에서 분리하여 구조를 가볍게 만들기
- 기능 변경: 없음

## 변경 파일
- `style.css`
- `css/web.css` 신규 추가
- `index.html`
- `sw.js`
- `sw-update.js`
- `app.js`
- `AUDIT_V4-22.md` 신규 추가

## 분리 내용
`style.css` 안에 있던 웹사이트 전용 CSS 중 독립성이 높은 항목을 `css/web.css`로 이동했다.

분리 대상:
- 웹사이트 탭 카테고리별 색상
- 즐겨찾기 탭 스타일
- 탭 카운트 배지
- 웹사이트 카드 기본 스타일
- 웹사이트 카드 상단/본문/하단 스타일
- 웹사이트 사제찾기 카드 일부 스타일
- 나의 교구 배지 일부 스타일
- 즐겨찾기 버튼 스타일
- 웹사이트 빈 목록 표시 스타일
- iOS 웹사이트 카드 밀도 보정 일부
- 웹사이트 카드 글자 크기 연동 일부

## 의도적으로 남긴 항목
아래 항목은 다른 화면과 공유되거나 cascade 순서가 민감하여 이번 단계에서 이동하지 않았다.

- `#web-view`, `#trail-view`, `#qna-view` 공유 모듈 레이아웃
- 웹사이트/순례길 공유 탭 구조
- 웹사이트/순례길 공유 목록 배경/스크롤 구조
- iOS 공통 헤더/탭 밀도 보정
- 글자 크기 조절 공통 규칙
- 커버 버튼의 웹사이트 바로가기 디자인
