# 한티가는길 독립 PWA v105

## 이번 변경
- 첨부된 모바일 UI를 참고해 전체 디자인을 파란색·흰색 중심으로 단순화
- 커버의 번호·장식·긴 설명을 줄이고 아이콘 중심의 4개 메뉴로 재구성
- 길안내와 한티가는길 안내를 넓은 주요 카드로, 숙박과 홈페이지를 작은 카드로 배치
- 안내 화면의 진한 남색·금색·베이지 계열을 제거하고 흰색 카드와 파란 포인트로 통일
- 코스 선택 화면을 단순한 상단바와 한 줄 코스 카드 구조로 정리
- 전체코스 버튼을 파란색 주요 버튼으로 강조하고 각 코스 카드의 빨간 점 장식 제거
- 지도 상단바, 내 위치 버튼, 하단 진행 정보와 조작 버튼을 같은 디자인 체계로 정리
- 기존 전체·코스별 지도, 위치 추적, 따라가기, 숙박 예약, 홈페이지, 카카오맵 교통 연결 기능 유지
- 한티순교성지 → 동대구역은 카카오맵의 한티순교성지 본 장소를 출발지로 유지
- PWA 테마 색상과 캐시를 v105로 갱신

## 커버 메뉴
1. 길안내 — 전체·코스별
2. 한티가는길 안내 — 코스·출발지·교통
3. 숙박 — 소울스테이 예약
4. 홈페이지 — 공식 사이트
