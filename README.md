# 한티가는길 독립 PWA v99

## 이번 변경
- 전국 순례길 목록을 제거하고 한티가는길 전용 앱으로 분리
- 첫 화면을 확장 가능한 메뉴형 커버로 변경
- 커버 메뉴 1: 길안내 — 전체코스 및 1~5코스별 선택
- 커버 메뉴 2: 숙박 — 추후 자료 추가를 위한 준비 화면
- 커버 메뉴 3: 홈페이지 — 한티가는길 공식 홈페이지 연결
- 기존 한티가는길 GPX 지도, 코스 경계, 위치 추적, 따라가기, 임시 복원 기능 유지
- 다른 교구 경로 데이터와 GPX를 제거해 앱 용량과 초기 로딩 범위를 축소
- PWA 이름, 설명, 캐시를 한티가는길 전용으로 변경

## 유지된 핵심 기준
- 전체지도 코스 색상은 공식 코스 경계에서 전환
- 1코스는 1-1부터 2-1까지 유지
- 개별 코스는 출발·도착 표시
- 내 위치 이동 확인, 정방향·역방향, 경로 따라가기 지원
- 8시간 이상 미사용 시 임시 내비게이션 상태 초기화
