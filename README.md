# 한티가는길 독립 PWA v101

## 이번 변경
- 커버에 `한티가는길 안내` 메뉴 추가
- 커버 메뉴 순서: 1 길안내 / 2 한티가는길 안내 / 3 숙박 / 4 홈페이지
- 공식 1~5코스 구간과 총거리 45.6km 안내
- 코스별 출발지와 주소 안내
- 5코스 출발지는 가산산성 진남문, 스탬프 위치는 진남문 주차장으로 구분
- 왜관역에서 가실성당까지 농어촌버스 9번·20번·21번 안내
- 안내 화면 전용 뒤로가기와 브라우저 뒤로가기 처리
- 기존 지도·위치 추적·따라가기·숙박 예약·홈페이지 연결 기능 유지
- PWA 캐시를 v101로 갱신
